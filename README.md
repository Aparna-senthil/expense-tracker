# Expense Tracker

This project is developed using Python and CSV file handling.

## Features
- Add Expense (Date, Category, Amount)
- View All Expenses
- Calculate Total Expense
- Category-wise Expense Summary
- Menu-driven program

## Technologies Used
- Python
- CSV Module
- File Handling
- Dictionary (for category-wise calculation)

## Description
This application stores expense records in a CSV file and performs analysis such as total expense and category-wise totals using Python.
